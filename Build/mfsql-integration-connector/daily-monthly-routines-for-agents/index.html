Daily Monthly Routines for Agents¶

There are a number of procedures that should be considered for inclusion in daily or monthly maintenance routines in SQL Agent Jobs.

  • spMFDeleteHistory :  this procedure will automatically delete history records from the logging tables that is older than the specified date.

  • spMFUpdateAllncludedInAppTables: this procedure will perform a daily update of all the class tables.
