MFvwAuditSummary¶

Purpose¶

To view a summary of AuditHistory by Flag and Class

Examples¶

Select * from MFAuditHistory

show audit history for a specific class

Select ah.*, mc.name from MFAuditHistory ah
inner join MFClass mc
on ah.Class = mc.mfid
Where mc.name = 'Customer'
